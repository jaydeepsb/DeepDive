How to create slides using Jupyter Notebook¶

by Jaydeep Belapure @DeepDive

Goto homepage

You need to install package called RISE¶

  • Install RISE using following command in conda command prompt
pip install rise

Install extensions, which will come in handy later on.¶

pip install jupyter_contrib_nbextensions
jupyter contrib nbextension install --user
pip install jupyter_nbextensions_configurator
jupyter nbextensions_configurator enable --user

Goto homepage

Check¶

After that refresh Jupyter notebook tab. You should be able to see

  • Enter/Exit RISE slideshow button in the toolbar.
  • Under Edit -> nbextensions config

Goto homepage

create an example code and slides¶

In [4]:
from numpy import arange
from pylab import plot, figure
from IPython.display import display, Math

x = arange(10)
y = x**2

display(Math('y = x^2'))
$\displaystyle y = x^2$

Using nbagg as matplotlib backend will display interactive plots.¶

In [5]:
%matplotlib nbagg
plot(x,y,"o-")

Set Slide Type¶

In the toolbar, you will see a new icon to enter/exit RISE slideshow mode. For each cell, explore different "Slide Types" (in the top-right corner) from the drop down menu.
